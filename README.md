🚀 Aprendendo Node.js + SQL

Repositório dedicado ao estudo de Back-end, focando na integração entre o ecossistema Node.js e bancos de dados relacionais.

🛠️ O que eu já implementei:
- **Integração SQL:** Conexão do Node.js com MySQL/PostgreSQL usando drivers nativos.
- **CLI (Command Line Interface):** Captura de inputs do usuário via terminal para persistência de dados.
- **Segurança e Ambiente:** Gerenciamento de variáveis de ambiente com `.env` e controle de versão ignorando dependências pesadas (`.gitignore`).
- **NPM:** Gestão de pacotes e dependências do projeto.
- **Sistema cadastro baseado em CRUD simples:** Um sistema de cadastro capaz de Visualizar, Atualizar, Cadastrar e Deletar usuarios de uma tabela do banco de dados. 

📦 Tecnologias:
- Node.js
- SQL (MySQL)
- Bibliotecas `mysql2` e `prompt-sync`
